Popup Boxes

Popup boxes are windows that pop out of the screen, appearing before a page loads, as an effect of an event, or before leaving a page. JavaScript has three types: alert boxes, user-input prompt boxes, and confirm boxes. They are useful for warnings, gathering user input, or verifying actions.

User-Input Prompt Box

This is used when data input is required before entering a page. Users must select “OK” or “Cancel.” If they choose “OK,” the prompt box returns their input; otherwise, it returns null.

Confirm Box

The confirm box asks the user to verify or accept something. They must choose “OK” or “Cancel.” If they select “OK,” it returns true; otherwise, it returns false.
