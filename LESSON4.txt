        <font style="font-size: 30px;"><i>Loops</i></font><br>
        <p style="padding: 5px;">
          There are times when you want a block of code repeated over and over again until a certain condition is satisfied. This can be done by creating a loop. A loop is a repetitive cycle of a block of code until the condition is met.<br><br>
          In JS, there are a variety of loops:<br>
          1. <b>For loop</b> – loops through a block of code within a specified number of times.<br>
          2. <b>While loop</b> – loops through a block of code until the condition is satisfied.<br>
          3. <b>Do-while loop</b> – like the while loop, but it will execute the set of codes at least once.<br>
          4. <b>For-in loop</b> – loops through the elements of an array.<br>
          <br><br>
          <i>Note:</i><br>
          If you want to exit from the loop after a condition has been met, use the <b>break</b> statement. This statement will immediately break the loop and continue executing codes after it. The <b>continue</b> statement will break the current loop iteration and continue to the next value.
        </p>

        <p><span style="color: #bda196;">For Loop:</span> Used when you know how many times the block of code must be iterated.</p><br>
        <p><span style="color: #bda196;">While Loop:</span> Used when you know when the condition will be met. It loops until the condition is met.</p><br>
        <p><span style="color: #bda196;">Do-While Loop:</span> Works like the while loop, except that the code will be executed once before checking the condition.</p><br>
        <p><span style="color: #bda196;">For-in Loop:</span> Loops through the elements of an array or an object.</p><br><br>
        <p><span style="color: #bda196;">Break and Continue statements:</span> The break statement stops the loop once a specified condition becomes true.</p><br>
        <p><span style="color: #bda196;">Continue statement:</span> The continue statement skips the current iteration and moves to the next value.</p><br>
